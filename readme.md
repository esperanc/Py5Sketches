# Programas de demonstração para o curso de Computação Visual 2026

Estes sketches foram produzidos como parte do material apresentado em sala de aula nona disciplina de Computação Visual ministrada na UFRJ em 2026.

Estes sketches podem ser executados diretamente a partir de um índice visual em https://esperanc.github.io/Py5Sketches. Eles foram criados no ambiente Py5Script, também disponível no github em https://esperanc.github.io/Py5Script.
